twilight-emacs by inkel

My custom Twilight Theme for Emacs

My customized version crafterm Twilight Them for Emacs See the original here: https://github.com/crafterm/twilight-emacs

Dependencies

Emacs, of course ;)

Install

Snapshot installation ================ Download the file in one of your `load-path` directories: wget -O ~/.emacs.d/site-listp/color-theme-twilight.el https://github.com/inkel/twilight-emacs/raw/master/color-theme-twilight.el And add to your Emacs initialization file: (require 'color-theme) (color-theme-initialize) (load-file "~/.emacs.d/site-lisp/color-theme-twilight.el") (color-theme-twilight)
